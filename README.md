# C++ Friend Function Demonstration

A concise C++ program that illustrates how friend functions work by granting external functions access to private class members.  
This example uses a `Box` class with a private attribute accessed via an external `printLength()` friend function.

---

## Features
- Demonstrates the use of `friend` keyword in C++  
- Allows external functions to access private class data  
- Simple and easy-to-understand OOP demonstration  
- Showcases encapsulation and controlled access  

---

## Concepts Used
- Classes & Objects  
- Private Data Members  
- Friend Function (`friend void printLength(Box)`)  
- Encapsulation & Access Control  

---

## How to Run
1. Save the program file as `main.cpp`  
2. Compile the program using:  
   ```bash
   g++ main.cpp -o box_friend
